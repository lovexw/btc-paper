# 比特币白皮书 - 中文详解版

> Bitcoin: A Peer-to-Peer Electronic Cash System - 详细注释和互动演示

## 📖 项目简介

这是一个比特币白皮书的中文详解网站，旨在用通俗易懂的方式解释区块链技术的核心概念。无论你是第一次接触比特币的新人，还是想深入学习的开发者，都能在这里找到有价值的内容。

### ✨ 特色功能

- 📝 **完整翻译**：比特币白皮书全文中文翻译
- 🎯 **通俗解释**：每个技术概念都配有新手友好的解释
- 💡 **生活类比**：用日常生活中的例子帮助理解复杂概念
- 💻 **互动演示**：
  - 哈希函数计算器
  - 挖矿模拟器
  - 数字签名演示
  - 区块链可视化
  - 篡改检测演示
- 📚 **术语表**：完整的区块链术语解释
- 🎨 **现代设计**：清晰、美观、响应式的用户界面

## 🚀 快速开始

### 本地运行

1. 克隆仓库：
```bash
git clone <repository-url>
cd <repository-folder>
```

2. 使用任何Web服务器打开 `index.html`，例如：
```bash
# 使用Python
python -m http.server 8000

# 使用Node.js
npx http-server

# 或直接在浏览器中打开 index.html
```

3. 在浏览器中访问 `http://localhost:8000`

## 🎓 内容概览

### 白皮书章节

1. **简介** - 了解为什么需要比特币
2. **交易** - 电子货币如何流转
3. **时间戳服务器** - 如何证明交易发生的时间
4. **工作量证明** - 比特币的核心安全机制
5. **网络** - 系统如何运作
6. **激励机制** - 为什么有人愿意维护网络
7. **回收磁盘空间** - 如何节省存储空间
8. **简化支付验证** - 轻量级钱包的原理
9. **价值的组合与分割** - 如何处理不同金额
10. **隐私** - 如何保护用户隐私
11. **计算** - 攻击者成功的概率分析
12. **结论** - 比特币系统总结

### 互动演示

- **哈希函数演示**：实时计算SHA-256哈希，观察输入变化如何影响输出
- **挖矿模拟器**：体验比特币挖矿的过程，调整难度观察所需算力
- **数字签名**：生成密钥对，签名和验证消息，测试篡改检测
- **区块链可视化**：创建区块链，观察区块如何链接，测试篡改检测

## 🎨 设计理念

本网站的设计参考了现代Web应用的最佳实践：

- **清晰的层次结构**：内容分级明确，易于导航
- **视觉一致性**：统一的配色方案和排版
- **交互反馈**：所有操作都有明确的视觉反馈
